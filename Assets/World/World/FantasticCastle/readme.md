Thank you for purchasing the Fantastic Medieval Castle Pixel Art Set.

The Sample environment has two different lightning setups. The Day and The Night.

There is simple scene manager script where you can toggle the day and the night lighting in the editor.
The script has 2 different GameObjects and all of the backgrounds and different global lighting is saved as child game objects. 

Pressing the toggle key, will disable the main game object (with all of its child objects on the scene) and enable the other main game object and vice versa.
